# Pilot Playbook — Hybrid AI Inbox (3–4 weeks) (2025-08-19)

**Design**: Cohorts A (new), B (reclaimed), C (existing control). Arms: Human vs AI‑first (HITL) vs Hybrid time‑split. Sample: ≥10 creators.
**KPIs**: RPM, ARPPU, tips/PPV mix, AOV, whale count, p50/p95 latency, refunds/blocks, policy violations, creator NPS.
**Runbook**: Inbox integration → routing → persona packs → safety/policy checker → human queue → QA rubric → analytics.
**Targets**: Week 4 GM ≥55%, payback ≤3 mo, incidents <1%. **Stops**: RPM delta < −10% for 2 weeks; refunds >3%; incidents ≥1%.
